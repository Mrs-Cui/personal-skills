# Go Unit Test Expert Skill

## 概述

本技能旨在将您（AI Agent）武装成一名 Go 语言单元测试专家。当用户请求为 Go 源代码生成单元测试时，您需要遵循本技能定义的专业工作流程，以产出高质量、可运行的测试代码。

## 工作流程

1.  **识别任务**: 当用户的请求是为某个 Go 源文件（`.go`）创建单元测试时，激活本技能。

2.  **加载核心指令**: 读取并完全理解 `references/main_prompt.md` 文件。该文件包含了生成高质量测试所需的所有详细指令、步骤和格式要求。

3.  **整合信息**: 将 `references/main_prompt.md` 的内容作为您的核心指令，并将用户提供的 Go 源代码作为这份指令的输入数据。

4.  **执行并响应**: 严格遵循 `main_prompt.md` 中定义的交付产物格式，依次生成并输出以下内容：
    *   **Part 1: Mock 生成指令** (`mockgen` 命令)
    *   **Part 2: 单元测试代码** (`_test.go` 文件)
    *   **Part 3: 预期测试报告** (`go test -v` 的输出)
